# Calculator Program

A comprehensive college project calculator that performs various mathematical operations and conversions.

## Description

This calculator program is designed to handle multiple types of calculations and conversions in a user-friendly interface. It's a feature-rich tool suitable for everyday mathematical needs, including basic arithmetic, statistical calculations, and unit conversions.

### Features

- **Basic Arithmetic Operations**
  - Addition (+)
  - Subtraction (-)
  - Multiplication (*)
  - Division (/)
  
- **Advanced Calculations**
  - Average calculation
  - VAT calculation
  
- **Unit Conversions**
  - Temperature conversion (Celsius, Fahrenheit)

#### **Prerequisites**
- Java Development Kit (JDK) 8 or higher installed
- IntelliJ IDEA Community Edition or Ultimate Edition
- Swing UI Designer plugin (should be pre-installed or available in IntelliJ marketplace)

#### **Method 1: Using IntelliJ IDE (Recommended)**

1. **Open the Project**
   - Launch IntelliJ IDEA
   - Click `File` → `Open`
   - Navigate to the calculator project folder and select it

2. **Build the Project**
   - Go to `Build` → `Build Project` (or press `Ctrl+F9`)
   - Wait for the build to complete successfully

3. **Run the Application**
   - Locate the main class file (usually named `Main.java`, `Calculator.java`, or `App.java`)
   - Right-click on the main class file in the Project Explorer
   - Select `Run 'ClassName'` (or press `Shift+F10`)
   - The calculator GUI window will open
