# Calculator Program

A college project calculator that performs various mathematical operations and conversions.

## Description

This calculator program is designed to handle multiple types of calculations and conversions in a user-friendly interface. It's a tool suitable for everyday mathematical needs, including basic arithmetic, statistical calculations, and unit conversions.

### Features

- **Basic Arithmetic Operations**
  - Addition (+)
  - Subtraction (-)
  - Multiplication (*)
  - Division (/)
  
- **Advanced Calculations**
  - Average calculation
  - VAT calculation
  
- **Unit Conversions**
  - Temperature conversion (Celsius, Fahrenheit)

#### **Prerequisites**
- Java Development Kit (JDK) 8 or higher installed
- IntelliJ IDEA Community Edition or Ultimate Edition
- Swing UI Designer plugin (should be available in IntelliJ marketplace)
